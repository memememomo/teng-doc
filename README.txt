* 概要

Tengに関して、有用なドキュメントをまとめるプロジェクトです。
http://walf443.github.com/teng-doc/

* 目的

Tengを使ったプロジェクトに参加する際に、このドキュメントをひととおり読みつつ自習してもらうことでTengを使うのに十分な知識を効率的にみにつけてもらえるようなドキュメントを作ることを目的としています。

* 執筆に参加したい方は

irc://#dbix-skinny@irc.perl.org のwalf443まで

あるいは、forkしてpull requestください

gh-pagesのbuildがウザいのでコラボに入らない方が快適かもしれません。

* buildする環境について

- spinxが必要です

** 構築方法

1. pythonbrewをいれpythonをbuildします
2. pip install sphinxします
3. make html
